## 4. C php v  Tnh nng c't li ca MQL5

#### 4.1.2. Qun l b nh: Stack, Heap v  cc cm by ph-  bin

Trong MQL5, b nh 'c qun l ch yu thng qua hai khu vc: Stack v  Heap. Vic hiu s khc bit gia chng v  cch chng 'c s dng l  rt quan trng ' trnh r r b nh v  cc l-i khc [8].

*   **Stack:** B nh Stack 'c s dng cho cc bin cc b, tham s' h m v  'a ch tr v. Vic cp pht v  gii phng b nh trn Stack di...n ra t 'ng v  rt nhanh chng. Kch thc ca Stack thng b gii hn v  vic s dng qu nhiu b nh Stack c th dn 'n l-i tr n Stack (Stack Overflow) [8].
*   **Heap:** B nh Heap 'c s dng cho vic cp pht b nh 'ng, tc l  b nh 'c cp pht trong thi gian chy ca chng trnh. Cc ''i tng 'c to bng ton t `new` s 'c cp pht trn Heap. B nh trn Heap khng t 'ng 'c gii phng khi h m kt thc; thay v o ', lp trnh vin phi t mnh gii phng n bng ton t `delete` khi khng cn s dng. Nu khng gii phng b nh ' cp pht trn Heap, s xy ra r r b nh (Memory Leak), dn 'n vic chng trnh tiu th ng y c ng nhiu b nh v  cu'i cng c th gy ra l-i hoc treo chng trnh [8].

**Cc cm by ph-  bin trong qun l b nh:**

*   **R r b nh:** Xy ra khi b nh 'c cp pht trn Heap nhng khng bao gi 'c gii phng. iu n y thng xy ra khi ''i tng 'c to bng `new` nhng khng c `delete` tng ng [8].
*   **Truy cp b nh sau khi gii phng (Use-after-free):** Xy ra khi chng trnh c' gng truy cp v o mt vng b nh ' 'c gii phng. iu n y c th dn 'n h nh vi khng xc 'nh hoc l-i chng trnh [8].
*   **Truy cp b nh ngo i gii hn (Out-of-bounds access):** Xy ra khi chng trnh c' gng 'c hoc ghi v o mt v tr b nh nm ngo i phm vi ca mt mng hoc ''i tng. iu n y c th l m hng d liu hoc gy ra l-i bo mt [8].

 qun l b nh hiu qu trong MQL5, cn tun th cc nguyn tc sau:

*   Lun gii phng b nh 'c cp pht bng `new` bng cch s dng `delete` khi ''i tng khng cn cn thit.
*   S dng cc con tr thng minh (nu c trong cc th vin bn ngo i hoc t trin khai) ' t 'ng ha vic qun l b nh.
*   Cn thn khi l m vic vi mng v  con tr ' trnh truy cp ngo i gii hn.
*   S dng trnh g l-i (debugger) ' pht hin v  khc phc cc vn ' lin quan 'n b nh.

#### 4.1.3. Truyn bin: Theo gi tr v  theo tham chiu

Trong MQL5, cc bin c th 'c truyn v o h m theo hai cch chnh: theo gi tr (by value) v  theo tham chiu (by reference) [1].

*   **Truyn theo gi tr (By Value):** Khi mt bin 'c truyn theo gi tr, mt bn sao ca bin ' s 'c to v  truyn v o h m. Mi thay '- i ''i vi bn sao bn trong h m s khng nh hng 'n bin g'c bn ngo i h m. y l  phng php truyn mc 'nh cho cc kiu d liu c bn (s' nguyn, s' thc, boolean, v.v.) [1].

    ```mq5
    void ModifyValue(int val)
    {
        val = val * 2;
        Print("Inside function (by value): ", val);
    }

    void OnStart()
    {
        int myVar = 10;
        Print("Before function call: ", myVar);
        ModifyValue(myVar);
        Print("After function call: ", myVar); // myVar remains 10
    }
    ```

*   **Truyn theo tham chiu (By Reference):** Khi mt bin 'c truyn theo tham chiu, 'a ch b nh ca bin g'c s 'c truyn v o h m, ch khng phi l  mt bn sao. iu n y c ngha l  mi thay '- i ''i vi bin bn trong h m s trc tip nh hng 'n bin g'c bn ngo i h m. Truyn theo tham chiu 'c thc hin bng cch s dng t kha `&` trc tn tham s' trong khai bo h m. Phng php n y thng 'c s dng cho cc ''i tng ln (v d: cu trc, lp, mng) ' trnh chi ph sao chp v  cho php h m sa '- i d liu g'c [1].

    ```mq5
    void ModifyReference(int& val)
    {
        val = val * 2;
        Print("Inside function (by reference): ", val);
    }

    void OnStart()
    {
        int myVar = 10;
        Print("Before function call: ", myVar);
        ModifyReference(myVar);
        Print("After function call: ", myVar); // myVar becomes 20
    }
    ```

    'i vi cc ''i tng phc tp nh mng v  chu-i, MQL5 thng truyn chng theo tham chiu mt cch ngm 'nh hoc cung cp cc c ch ' thao tc trc tip vi d liu g'c, gip t'i u ha hiu sut. Vic la chn phng php truyn bin ph hp l  rt quan trng ' 'm bo tnh 'ng 'n ca chng trnh v  hiu qu s dng b nh.

### 4.3. Lp trnh hng ''i tng (OOP) trong MQL5

MQL5 l  mt ngn ng lp trnh hng ''i tng (OOP) 'y ', h- tr cc khi nim c't li ca OOP nh 'ng gi (encapsulation), k tha (inheritance) v  'a hnh (polymorphism). Vic p dng OOP trong MQL5 gip cc nh  pht trin xy dng cc h th'ng giao dch phc tp mt cch c t-  chc, d... bo tr v  m rng.

#### 4.3.1. Cc khi nim OOP: ng gi, K tha, a hnh

*   **ng gi (Encapsulation):** L  nguyn tc che giu d liu v  h nh vi bn trong mt lp, ch cho php truy cp thng qua cc phng thc cng khai (public methods). iu n y gip bo v tnh to n vn ca d liu v  gim s ph thuc gia cc phn khc nhau ca chng trnh. Trong MQL5, bn s dng cc t kha `public`, `protected`, `private` ' kim sot quyn truy cp [1].

    ```mq5
    class CAccountInfo
    {
    private:
        string m_accountName; // Private member variable
        double m_balance;

    public:
        // Constructor
        CAccountInfo(string name, double balance)
        {
            m_accountName = name;
            m_balance = balance;
        }

        // Public method to access account name
        string GetAccountName() const { return m_accountName; }

        // Public method to get balance (read-only)
        double GetBalance() const { return m_balance; }

        // Public method to update balance (controlled access)
        void UpdateBalance(double newBalance)
        {
            if (newBalance >= 0) // Simple validation
            {
                m_balance = newBalance;
            }
            else
            {
                Print("Invalid balance value.");
            }
        }
    };
    ```

*   **K tha (Inheritance):** Cho php mt lp (lp con hoc lp dn xut) k tha cc thuc tnh v  phng thc t mt lp khc (lp cha hoc lp c s). iu n y thc 'y kh nng ti s dng m v  to ra mt h th'ng phn cp cc lp. MQL5 h- tr k tha 'n [1].

    ```mq5
    // Base class for a generic trading instrument
    class CTradingInstrument
    {
    protected:
        string m_symbol;
        double m_currentPrice;

    public:
        CTradingInstrument(string symbol) : m_symbol(symbol) {}

        virtual void UpdatePrice() = 0; // Pure virtual function
        string GetSymbol() const { return m_symbol; }
        double GetCurrentPrice() const { return m_currentPrice; }
    };

    // Derived class for a Forex instrument
    class CForexInstrument : public CTradingInstrument
    {
    public:
        CForexInstrument(string symbol) : CTradingInstrument(symbol) {}

        void UpdatePrice() override
        {
            m_currentPrice = SymbolInfoDouble(m_symbol, SYMBOL_ASK);
            Print("Forex Instrument ", m_symbol, " price updated to ", m_currentPrice);
        }
    };

    // Derived class for a Stock instrument
    class CStockInstrument : public CTradingInstrument
    {
    public:
        CStockInstrument(string symbol) : CTradingInstrument(symbol) {}

        void UpdatePrice() override
        {
            m_currentPrice = SymbolInfoDouble(m_symbol, SYMBOL_LAST);
            Print("Stock Instrument ", m_symbol, " price updated to ", m_currentPrice);
        }
    };
    ```

*   **a hnh (Polymorphism):** L  kh nng cc ''i tng thuc cc lp khc nhau c th 'c x l thng qua mt giao din chung (thng l  con tr hoc tham chiu 'n lp c s). iu n y cho php vit m linh hot hn, c th hot 'ng vi nhiu loi ''i tng m  khng cn bit kiu c th ca chng ti thi 'im bin dch. a hnh trong MQL5 'c h- tr thng qua cc h m o (virtual functions) v  con tr/tham chiu 'n lp c s [1].

    ```mq5
    void ProcessInstrument(CTradingInstrument* instrument)
    {
        instrument.UpdatePrice(); // Calls the appropriate UpdatePrice based on actual object type
    }

    void OnStart()
    {
        CForexInstrument* eurUsd = new CForexInstrument("EURUSD");
        CStockInstrument* aapl = new CStockInstrument("AAPL");

        ProcessInstrument(eurUsd);
        ProcessInstrument(aapl);

        delete eurUsd;
        delete aapl;
    }
    ```

#### 4.3.2. Cc Design Pattern ph-  bin: Strategy, Singleton, Factory (vi v d code)

Design Patterns l  cc gii php ' 'c kim chng cho cc vn ' thit k phn mm lp 'i lp li. Vic p dng Design Patterns trong MQL5 gip ci thin cu trc m, kh nng ti s dng v  bo tr. Di 'y l  mt s' Design Pattern ph-  bin v  cch chng c th 'c p dng trong MQL5 [9]:

*   **Strategy Pattern:**
    *   **Mc 'ch:** nh ngha mt h cc thut ton, 'ng gi tng thut ton v  l m cho chng c th hon '- i cho nhau. Strategy Pattern cho php thut ton thay '- i 'c lp vi cc client s dng n.
    *   **ng dng trong MQL5:** Rt ph hp ' trin khai cc chin lc giao dch khc nhau (v d: chin lc theo xu hng, chin lc 'o chiu, chin lc giao dch tin tc) m  khng cn sa '- i m chnh ca Expert Advisor. M-i chin lc 'c 'ng gi trong mt lp ring bit.

    ```mq5
    // Abstract Strategy Interface
    class ITradingStrategy
    {
    public:
        virtual void Execute() = 0;
    };

    // Concrete Strategy 1: Trend Following
    class CTrendFollowingStrategy : public ITradingStrategy
    {
    public:
        void Execute() override
        {
            Print("Executing Trend Following Strategy...");
            // Implement trend following logic here
            // e.g., use moving averages, ADX, etc.
        }
    };

    // Concrete Strategy 2: Mean Reversion
    class CMeanReversionStrategy : public ITradingStrategy
    {
    public:
        void Execute() override
        {
            Print("Executing Mean Reversion Strategy...");
            // Implement mean reversion logic here
            // e.g., use Bollinger Bands, RSI, etc.
        }
    };

    // Context Class: Expert Advisor that uses a strategy
    class CExpertAdvisor
    {
    private:
        ITradingStrategy* m_strategy;

    public:
        CExpertAdvisor() : m_strategy(NULL) {}

        void SetStrategy(ITradingStrategy* strategy)
        {
            m_strategy = strategy;
        }

        void OnTick()
        {
            if (m_strategy != NULL)
            {
                m_strategy.Execute();
            }
            else
            {
                Print("No trading strategy set.");
            }
        }

        ~CExpertAdvisor()
        {
            // Clean up strategy if owned by EA
            if (m_strategy != NULL)
            {
                delete m_strategy;
                m_strategy = NULL;
            }
        }
    };

    // Usage Example
    CExpertAdvisor* myEA;

    int OnInit()
    {
        myEA = new CExpertAdvisor();
        // Choose a strategy based on external parameters or market conditions
        if (true) // Example condition
        {
            myEA.SetStrategy(new CTrendFollowingStrategy());
        }
        else
        {
            myEA.SetStrategy(new CMeanReversionStrategy());
        }
        return(INIT_SUCCEEDED);
    }

    void OnTick()
    {
        myEA.OnTick();
    }

    void OnDeinit(const int reason)
    {
        if (myEA != NULL)
        {
            delete myEA;
            myEA = NULL;
        }
    }
    ```

*   **Singleton Pattern:**
    *   **Mc 'ch:** m bo rng mt lp ch c mt th hin (instance) duy nht v  cung cp mt 'im truy cp to n cc 'n th hin '.
    *   **ng dng trong MQL5:** Hu ch cho cc ''i tng cn c mt th hin duy nht trong to n b chng trnh, chng hn nh trnh ghi nht k (logger), qun l cu hnh to n cc, hoc kt n'i 'n mt t i nguyn bn ngo i duy nht.

    ```mq5
    // Singleton class for managing global settings
    class CGlobalSettings
    {
    private:
        static CGlobalSettings* m_instance; // Static pointer to the single instance
        string m_logFilePath;
        double m_slippage;

        // Private constructor to prevent direct instantiation
        CGlobalSettings()
        {
            m_logFilePath = "\\Logs\\MyEA.log";
            m_slippage = 0.5; // Default slippage
            Print("CGlobalSettings instance created.");
        }

        // Private copy constructor and assignment operator to prevent copying
        CGlobalSettings(const CGlobalSettings& other) = delete;
        CGlobalSettings& operator=(const CGlobalSettings& other) = delete;

    public:
        // Static method to get the single instance
        static CGlobalSettings* GetInstance()
        {
            if (m_instance == NULL)
            {
                m_instance = new CGlobalSettings();
            }
            return m_instance;
        }

        // Public methods to access and modify settings
        string GetLogFilePath() const { return m_logFilePath; }
        void SetLogFilePath(string path) { m_logFilePath = path; }

        double GetSlippage() const { return m_slippage; }
        void SetSlippage(double slippage) { m_slippage = slippage; }

        // Destructor to clean up the instance (optional, for explicit cleanup)
        ~CGlobalSettings()
        {
            Print("CGlobalSettings instance destroyed.");
        }
    };

    // Initialize the static member outside the class definition
    CGlobalSettings* CGlobalSettings::m_instance = NULL;

    // Usage Example
    void OnStart()
    {
        // Get the single instance of CGlobalSettings
        CGlobalSettings* settings1 = CGlobalSettings::GetInstance();
        Print("Log File Path (settings1): ", settings1.GetLogFilePath());

        // Modify a setting through settings1
        settings1.SetSlippage(1.0);

        // Get the instance again (should return the same instance)
        CGlobalSettings* settings2 = CGlobalSettings::GetInstance();
        Print("Slippage (settings2): ", settings2.GetSlippage()); // Should be 1.0

        // No need to delete m_instance explicitly in OnStart for EAs/Indicators
        // as it's a global resource. It will be cleaned up on program termination.
        // If you need explicit cleanup, you can add a static DestroyInstance() method.
    }
    ```

*   **Factory Method Pattern:**
    *   **Mc 'ch:** nh ngha mt giao din ' to mt ''i tng, nhng ' cc lp con quyt 'nh lp n o s 'c khi to. Factory Method cho php mt lp tr hon vic khi to cho cc lp con.
    *   **ng dng trong MQL5:** Hu ch khi bn cn to cc ''i tng thuc cc lp khc nhau da trn mt 'iu kin hoc tham s', nhng bn mu'n tru tng ha qu trnh to ''i tng khi m client. V d, to cc loi lnh giao dch (Market Order, Pending Order) hoc cc loi ch bo khc nhau.

    ```mq5
    // Abstract Product: Base class for a Trading Strategy
    class CTradingStrategy
    {
    public:
        virtual string GetStrategyName() = 0;
        virtual bool ExecuteStrategy() = 0;
    };

    // Concrete Product 1: Trend Following Strategy
    class CTrendFollowingStrategy : public CTradingStrategy
    {
    public:
        string GetStrategyName() { return "Trend Following"; }
        bool ExecuteStrategy() { /* Logic for trend following */ Print("Executing Trend Following Strategy."); return true; }
    };

    // Concrete Product 2: Mean Reversion Strategy
    class CMeanReversionStrategy : public CTradingStrategy
    {
    public:
        string GetStrategyName() { return "Mean Reversion"; }
        bool ExecuteStrategy() { /* Logic for mean reversion */ Print("Executing Mean Reversion Strategy."); return true; }
    };

    // Abstract Creator: Defines the factory method
    class CStrategyCreator
    {
    public:
        virtual CTradingStrategy* CreateStrategy() = 0;

        void RunStrategy()
        {
            CTradingStrategy* strategy = CreateStrategy(); // Factory method call
            if (strategy != NULL)
            {
                Print("Running strategy: ", strategy.GetStrategyName());
                strategy.ExecuteStrategy();
                delete strategy;
            }
        }
    };

    // Concrete Creator 1: Creates Trend Following Strategy
    class CTrendFollowingCreator : public CStrategyCreator
    {
    public:
        CTradingStrategy* CreateStrategy() { return new CTrendFollowingStrategy(); }
    };

    // Concrete Creator 2: Creates Mean Reversion Strategy
    class CMeanReversionCreator : public CStrategyCreator
    {
    public:
        CTradingStrategy* CreateStrategy() { return new CMeanReversionStrategy(); }
    };

    // Usage Example
    void OnStart()
    {
        CStrategyCreator* trendCreator = new CTrendFollowingCreator();
        trendCreator.RunStrategy();
        delete trendCreator;

        CStrategyCreator* meanReversionCreator = new CMeanReversionCreator();
        meanReversionCreator.RunStrategy();
        delete meanReversionCreator;
    }
    ```
